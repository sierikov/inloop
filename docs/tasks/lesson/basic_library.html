A Basic Library

Your task is to write a basic library. It must be possible to add new books to the library. This task should help you with the following basic Java programming questions:

  1. How are classes combined to form a working program?
  2. Where does a Java program start?
  3. What are arrays and how can I use them?
  4. What does overriding a method mean?

The following three classes must be implemented:

  1. Book - represents a book in your library.
  2. Library - represents the library itself.
  3. HelloLibrary - starting class which contains the main() method; the entry point of the program.
The Book class

Objects of this class represent books. Each book has a String attribute title for the book’s title.

Whenever a new Book object is being created, which happens when the constructor of the class Book is called, a status message has to be printed. Assuming the book’s title is Java for Dummies, the message would be: 

Book Java for Dummies created.

The constructor of Book receives the title of the book to be created as parameter. That parameter is used to initialize the title attribute with the respective title of the book.

To make the title easier to access, the toString() method must be overridden. It returns a String representation of an object, in our case the book’s title. After overriding, one could print a book’s title with the following lines of code: 

// Create a book called 'Java for Dummies'
// and put it into the variable book
Book book = new Book("Java for Dummies");

// Print the title of the book
System.out.println(book);

The variable book, containing a reference to an object of type Book, can be used directly in the println() statement. Implicitly, Java calls the toString() method. If you do not overwrite the toString() method, a default implementation of toString() is executed and the object’s ID within the system is returned. To make the code above return the proper book title, you must override the method toString() in your class Book.

The Library class

Objects of this class represent libraries.

Whenever a Library object is being created, the following message must be printed:

Hello, I am a library, which can store up to 10 books!
Adding Books

It should be possible to add books (objects of type Book) to this library using the method add(Book book). This method has to print a message upon being called, too. If this method was called while the parameter book contained a Book object with the title "Java for Dummies", the following message would have to be printed: 

I added the book Java for Dummies!

The Book object passed to the method as argument must be stored in the Library object.

It should not be possible to add more than 10 books to a library. The following line must be printed if the library already holds 10 books and the add() method is called another time: 

The library is full!
Searching for Books

The method search(String title) should search the library for a book with a certain title and must return the first Book object with the given title. If the library does not contain any book with the title, null must be returned instead. Also, one of the following lines must be printed after a search has been performed:

  1. If a book with the given title has been found 
    The book with the title Java for Dummies exists in the library!
  2. If no book with the given title has been found
    The book with the title Java for Dummies does not exist in the library!

Of course, you have to use title of the respective book!

Hints

For now, you should use an array attribute to store the Book objects in a Library object. An array is a data structure that can hold a fixed amount of objects of the same type. Since the space in an array is limited and fixed, you have to pay attention to the limits when trying to access entries of an array. 

int[] array = new int[20];

This array can only store 20 values of type int (a primitive type for integers (whole numbers)). Each of these 20 entries of the array has an integer index. The first entry has the index 0, the second entry’s index is 1, the third’s is 2 and so on. The index of the last entry is 19 which is one less than the length of the array.

The following two lines of code show how you can write the value 7 to the first array entry and 3 to the last entry. 

array[0] = 7;
array[19] = 3;
The HelloLibrary Class

Every program requires an entry point where execution of the program begins. In Java, this is the main() method. The method’s signature must be exactly like below (except for the parameter’s name) in order to be a valid execution entry point. 

public static void main(String[] args) {
    // write some code here
}

In our case, the main() method should be put into the class HelloLibrary. It must solve the following tasks:

  1. Create an object of type Library and save it in a variable called lib.
  2. Create an object of type Book and save it in a variable called book1.
  3. Create an object of type Book and save it in a variable called book2.
  4. Add the two Book objects book1 and book2 to the library using the method add() of the Library object referenced by the variable lib.

Your program should print the following output if you have done everything correctly:

Hello, I am a library, which can store up to 10 books!
Book UML created.
Book Java2 created.
I added the book UML!
I added the book Java2!
