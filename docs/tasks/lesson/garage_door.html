Garage Door

A garage door (class GarageDoor) is controlled by a motor driven mechanism. It opens and closes the door.

To control the garage door, there is a control panel with one button to open the door (openDoor()) and another one to close it (closeDoor()). These buttons can be used at any time. Furthermore, the door is equipped with two end switches. Each of them sends a signal (stopper()) to the control if the opening or closing of the door has been completed.

The Door

In the state design pattern, we use an abstract state class (DoorState), which provides the design of all concrete states. The concrete states are represented by concrete classes that are subclasses of the abstract class DoorState. Implement the state pattern the way it was introduced in the lecture (as private inner classes of the context class GarageDoor)!

Remarks

In a real world scenario, the methods of the class Motor would contain the commands for controlling the motor. Since we do not have a real motor and this is only an example, you can fill them with some status reports printed to the console if you want to.
