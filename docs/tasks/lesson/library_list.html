Collections I

Until now, arrays have been the data structure of choice when we had to manage several objects of the same type. The purpose of this task is to change this circumstance. The Java API provides more elaborate data structures than arrays. In this task, we want to use collections from the Java collection framework (JCF) to implement a simple library program.

Task

Implement the classes in the model above with the following constraints!

  1. Use the List interface to manage the Book objects!
  2. As concrete List type, use either ArrayList or LinkedList! Write the code in such a way that you have to make as few changes as possible when it comes to switching the list type! Think of polymorphism to reach this goal!
  3. Use the static method Collections.binarySearch() to implement Library.searchForISBN()!
  4. Think about the advantages of an implementation with lists over an implementation with arrays!
