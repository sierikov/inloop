Java and Functions

Write a program which calculates the factorial of a given number. Call your class Functions (filename Functions.java) and create and implement the function

To solve this task follow the instructions given in the task description.

public static int factorial(int n) {
    // insert your implementation here
}

which calculates the factorial of n. Mind that the method must be static!

Add a main method to Functions and use it to calculate the factorial of 5. Print the result to the console.

The output should look like the following:

The factorial of 5 is 120.
Hints

The method public static void main(String... args) is the starting point of every Java program. So create the Functions.main(String... args) method with the given method parameters. You have to call Functions.factorial(int n) inside this method to start the calculation.

You can output the value of any variable of a primitive data type (bool, int, double etc.) or any String variable to the console with System.out.println(x).

Functions in Java are used and implemented the same way as in other programming languages or as they are known from mathematics. They can look like the following: 

public static float myFunction(int x, String text)
{
    return 1.0;
}

Usually, functions have a return type (they return a specific value after successful execution). The function above returns a floating point (type float) number.

Functions take a specific number of arguments (called parameters). It is possible that a function does not take any argument at all. Our example above takes an integer number (type int) and a String object as arguments. Additionaly, our function has the keyword static, which means that this function belongs to the class itself rather than an instance (object) of the class.
