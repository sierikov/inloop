String Processing

Write a Java class Leet (filename Leet.java) that contains the methods from task 1 and task 2 for the translation of normal language into leet language.

So, what is leet?

Leet (also known as 1337) is an Internet sociolect and a modification of written (in most cases English) text. Still, what does this mean? For example, some cool hacker-guy from 1980 would never write “cool program” but instead “k3wl pr0gr@m”.

Task 1

Implement the method

public static String toLeet(String normal)

which translates the given string normal into leet by performing the following transformations. Note that the order in which the conversions are performed is important.

Normal Leet
elite 1337
cool k3wl
! !!!11
ck xx
ers 0rz
er 0rz
en n
e 3
t 7
o 0
a @

Although there are plenty of other possible conversions, our lesson requires to implement only the ones in the table above. Consult the Wikipedia page of Leet for more possibilities.

Hints

The class String provides the method

String replace(String needle, String replacement)

that returns a (new) String object in which all occurrences of needle in the String the method is called on (also referred to as haystack) are replaced with replacement. Example: 

String hacker = "I am such a cool hacker!";
hacker = hacker.replace("cool", "crappy");

After that, hacker contains the value: "I am such a crappy hacker!". Instead of String.replace(String, String), you could use the method String.replaceAll(String regex, String replacement): 

hacker = hacker.replaceAll("cool", "crappy")

In this case, the result is exactly the same.

For more advanced programmers: String.replaceAll(...) uses regex as regular expression pattern to look for matches in the haystack while String.replace(...) just looks for exact literal matches. So in the end, String.replaceAll(...) is far more powerful.

Task 2

Implement the method

public static String[] allToLeet(String[] normals)

which converts all elements of the given String array normals to Leet and returns them in a new String array. Use a for loop in order to convert the elements of normals and to create the new array which is returned afterwards. Make use of your method toLeet(...) from task 1 for every single conversion.

Arrays

An array is a container for several values of the same type where each of these values has an index. The following code creates a new array that contains two String objects: 

String[] myArray = new String[2];

Each array element can be accessed through its index. The first element has the index 0 (zero).

  1. Writing into an array

    myArray[0] = "Hallo";
myArray[1] = "Banana-Milkshake";
myArray[2] = "...";               //  This produces an error (IndexOutOfBoundsException)
				  //  because the array only has two elements

  2. Reading from an array

    System.out.println(myArray[0]);

    writes “Hallo” to the console.

The length of an array can be accessed via myArray.length. See the Java API documentation for more details.

Loops

The syntax of the for loop in Java is the same as in any other C-like programming language:

for(initialization; loopCondition; increment/decrement)
{
    // my code
}

Example:

for(int i = 0; i < 10; i++)
{
    System.out.println("Loop " + i);
}

The code in this loop will be executed as long as i is smaller than 10. Starting at 0, the variable i is incremented by 1 each time the code inside the loop has been executed and the condition is still fulfilled. Therefore, the code is run ten times.

Task 3

"We are elite hackers!"

Thereafter, translate all elements of the array into leet with your method allToLeet(...) from task 1. Print the converted strings to the console.

You should get the following output: 

W3 @r3 l337 h@xx0rz!!!11
Strings

In Java, you can easily handle strings:

String myString1 = "Hello";

This creates a new String object.
With the concatenation operator + (plus sign) you can put strings together: 

String myString2 = "World!";
String result = myString1 + " " + myString2;

You can also put numbers into strings:

String number = "This is a one: " + 1;

Java automatically converts the int literal 1 into a String object containing "1" (without quotation marks) and concatenates it to the other string.
